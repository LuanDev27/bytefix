# ByteFix — Site institucional

Site da **ByteFix** (São Luís – MA): manutenção de impressoras, notebooks e desktops, e criação de sites profissionais.

Site estático (HTML + CSS + JavaScript, sem dependências e sem build).

## Estrutura

- `index.html` — site principal (serviços, preços, funil, portfólio ao vivo, depoimentos, FAQ)
- `demos.html` — galeria de exemplos de sites
- `demo-pizzaria.html` — exemplo: pizzaria / restaurante
- `demo-barbearia.html` — exemplo: barbearia
- `demo-estetica.html` — exemplo: estética / beleza
- `demo-academia.html` — exemplo: academia / personal
- `art-*.svg` — ilustrações de cada segmento

## Como rodar localmente

Basta abrir `index.html` no navegador. (As prévias ao vivo do portfólio funcionam melhor quando publicado, ex.: Vercel.)

## Publicar na Vercel

1. Importe este repositório em https://vercel.com/new
2. Framework Preset: **Other** (site estático, sem build)
3. Deploy — pronto.
